# Graphify 3D – Bird’s Eye View of Vela

This project uses **Graphify** and **3d-force-graph** to create an interactive 3D visualization of a complex graph structure. It provides a bird’s-eye view of vela, helping explore relationships, clusters, and connectivity in real time.

## 🧠 What it does

- Builds an interactive 3D force-directed graph
- Groups nodes into communities
- Lets you toggle visibility by community
- Provides real-time exploration of relationships
- Offers a clean bird’s-eye overview of complex structures

## 🛠️ Tech Stack

- Graphify (custom graph structure layer)
- 3d-force-graph (WebGL-based 3D rendering)
- D3.js (force simulations)
- Vanilla JavaScript / HTML / CSS
